# SGPA - Sistema de Gerenciamento de Projetos Acadêmicos

> [cite_start]**Nota:** Este é um projeto de fins estritamente acadêmicos, desenvolvido para consolidar conceitos de Engenharia de Software[cite: 9, 10].

## 📖 Visão Geral
[cite_start]O SGPA é uma aplicação web concebida para centralizar o cadastro, gerenciamento e acompanhamento de projetos acadêmicos[cite: 18]. [cite_start]O sistema visa facilitar o controle de orientadores, alunos e a evolução das atividades de pesquisa e extensão[cite: 19, 20, 21].

## 🚀 Funcionalidades (Escopo)
[cite_start]O projeto contempla o gerenciamento completo (CRUD) das seguintes entidades[cite: 26]:
* [cite_start]**Gestão de Alunos e Professores:** Cadastro e manutenção de perfis institucionais[cite: 36, 82].
* [cite_start]**Gestão de Projetos:** Criação de projetos com definição de prazos, orientadores e status automático[cite: 97, 107].
* [cite_start]**Módulo de Equipes:** Vinculação de alunos a projetos com definição de papéis (Participante ou Líder)[cite: 126, 130].
* [cite_start]**Tarefas e Entregas:** Controle de atividades por equipe, com suporte a envio de arquivos e logs de progresso[cite: 139, 153].
* [cite_start]**Relatórios:** Geração de relatórios de andamento com cálculo automático de produtividade[cite: 178, 184].

## 👥 Perfis de Usuário
* [cite_start]**Administrador:** Gestão global de cadastros, usuários e permissões de acesso[cite: 33].
* [cite_start]**Professor:** Orientação direta de projetos e gestão de tarefas/entregas das suas equipes[cite: 33].
* [cite_start]**Aluno:** Participação em projetos, atualização de progresso e submissão de entregas[cite: 33].

## 🛠️ Requisitos Técnicos e Qualidade
[cite_start]Seguindo as diretrizes de Engenharia de Software estabelecidas, o sistema adota[cite: 193]:
* [cite_start]**Arquitetura:** Implementação baseada no padrão **MVC**[cite: 193].
* [cite_start]**Segurança:** Autenticação com senhas criptografadas via **bcrypt** e sessões protegidas por cookies httpOnly[cite: 193].
* [cite_start]**Documentação:** Especificação da API seguindo o padrão **OpenAPI 3.0**[cite: 193].
* [cite_start]**Performance:** Consultas e inserções otimizadas para resposta em menos de 3 segundos[cite: 193].
* [cite_start]**Qualidade:** Garantia de funcionamento via testes automatizados de CRUD com **Selenium**[cite: 193].
